# Trust Atlas Style Guide

Visual identity and brand guidelines for Trust Atlas.

## Relationship to Trust Revolution

Trust Atlas is an open-source project from [Trust Revolution](https://trustrevolution.co).

- **Trust Revolution** asks: *"Why do we trust the systems we trust?"*
- **Trust Atlas** answers: *"How much do we actually trust them?"*

One is the **conversation**, the other is the **evidence**.

## Visual Identity

### Color Palette

```css
/* Background */
--slate-950: #020617

/* Text */
--slate-400: #94a3b8    /* Secondary */
--slate-300: #cbd5e1    /* Primary on dark */

/* Accent */
--amber-400: #fbbf24    /* Primary accent */
--amber-500: #f59e0b    /* CTA buttons */

/* Pillar Colors */
--blue-400: #60a5fa     /* Interpersonal trust */
--amber-400: #fbbf24    /* Institutional trust */
--emerald-400: #34d399  /* Governance quality */
```

### Typography

- **Display font**: Playfair Display (serif) — authority, trust
- **Body font**: Source Sans 3 (sans-serif) — data readability

### Design Principles

- Dark mode primary (slate-950 background)
- Cartographic/data-viz aesthetic
- Emphasis on charts, maps, numbers
- Serious but accessible tone

## Audience

| Audience | Tone | CTA |
